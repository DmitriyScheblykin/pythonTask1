Составить новый список чисел из двух переданных списков следующим образом. На нечётные позиции ставить элементы из первого списка, на чётные – из второго. В случае, если один список короче другого, то на недостающие позиции (неважно, четные или нечетные) ставить элементы из более длинного списка (т.е. итоговый список обязательно будет длиной, равной длине более длинного списка), например:

({ 1, 2, 3, 4 }, { 101, 102, 103, 104, 105, 106, 107 }) → { 1, 102, 3, 104, 105, 106, 107 }
